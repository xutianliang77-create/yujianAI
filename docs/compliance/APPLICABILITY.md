# 中国部署适用性清单（工程占位）

本文件不是法律意见。每一项必须由项目负责人和合规/法务确认后才能开启对应能力。

| 主题 | 当前结论 | 阻断条件 | 负责人 |
| --- | --- | --- | --- |
| PIPL/个人信息 | 适用，默认最小化与脱敏 | 数据分类、删除/导出证据未完成 | compliance-owner |
| 网络安全法/等保 | 需按客户部署形态评估 | 等保范围与测评计划未确认 | security-owner |
| ICP/增值电信 | 托管公网服务需评估 | 主体、地域、业务范围未确认 | legal-owner |
| 生成式 AI | Agent provider 按能力逐项评估 | provider 备案、内容安全和留痕未确认 | ai-governance-owner |
| SIP/录音 | 默认禁用 | 运营商、资质、告知、保留/删除闭环未完成 | telephony-owner |

所有未知项在 Gate 0/5/6/10 前保持阻断，不得通过配置默默启用。
